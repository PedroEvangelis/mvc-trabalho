# Consultas — Behaviors

## Proactive Analysis

### Defaults
- Listas ordenadas por data de matrícula (decrescente)

### Failure Modes
- Turma inexistente → erro
- Aluno inexistente → erro
- Turma sem alunos → lista vazia
- Aluno sem matrículas → lista vazia
