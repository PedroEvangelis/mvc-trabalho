# VISION — Clínica Veterinária

## Problema
Clínica veterinária pequena controla tudo em fichas de papel. Tutores ligam para saber o histórico do pet e não encontram. Animais com o mesmo nome são confundidos entre tutores diferentes, gerando registros no animal errado.

## Atores
- **Veterinário** — profissional que usa o sistema para registrar e consultar dados

## Métricas de sucesso
- Tutor encontra o histórico completo do animal em segundos
- Nenhuma consulta é registrada no animal errado
- Pagamento das consultas é controlado (PENDENTE / PAGO)
- Dados não se perdem mais com fichas físicas

## Fora de escopo
- Login com senha ou autenticação
- Relatórios financeiros ou fiscais
- Controle de estoque de medicamentos
- Agendamento de consultas
- Prescrição de receitas
