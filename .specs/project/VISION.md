# VISION — Oficina Mecânica

## Problema
Oficina mecânica anota tudo à mão em caderno. Quando um cliente volta, é preciso perguntar novamente qual é o carro, o que foi feito antes e quanto ele deve. Não há histórico confiável.

## Atores
- **Mecânico / Atendente** — usuário do sistema que gerencia clientes, veículos, serviços e ordens de serviço

## Métricas de sucesso
- Cliente é identificado rapidamente com todos os veículos
- Histórico completo de manutenções por veículo disponível em segundos
- Ordens de serviço com status claro (ABERTA / CONCLUIDA)
- Pagamento das OS controlado (PENDENTE / PAGO)

## Fora de escopo
- Login com senha
- Controle de estoque de peças
- Nota fiscal
- Agendamento de horários
